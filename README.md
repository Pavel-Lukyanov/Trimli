# UrlShortener

- **PostgreSQL** – база данных с указанным пользователем, паролем и базой (`.env` используется для конфиденциальных данных)
- **Redis** – кэш/очередь, защищён паролем

1. Копируйте файл `.env.example` и переименуйте копию в `.env`:
   ```bash
   cp .env.example .env
2. Заполните .env своими доступами
3. В своем дистрибутиве Linux (Ubuntu/Debian..) установите утилиту "make" командой
   ```bash
   sudo apt install make -y
4. Для запуска всех контейнеров, в консоли выполните команду
   ```bash
   make up
5. Для остановки всех контейнеров, в консоли выполните команду
   ```bash
   make down

## Список команд make

| Команда                     | Что делает                            |
| --------------------------- | ------------------------------------- |
| `make up`                   | Поднять все контейнеры                |
| `make build`                | Пересобрать контейнеры и запустить их |
| `make down`                 | Остановить все контейнеры             |
| `make restart`              | Перезапустить контейнеры              |
| `make logs`                 | Просмотр логов контейнеров            |
| `make ps`                   | Список запущенных контейнеров         |
| `make shell service=<name>` | Зайти в контейнер (например, `minio`) |
| `make clean`                | Удаляет контейнеры и volume'ы         |
| `make prune`                | Полная очистка Docker (внимательно!)  |
